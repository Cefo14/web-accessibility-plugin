A simple accessibility plugin alternative

Lightweight, fast and easy to integrate. Enhance web accessibility with customizable text options, visual filters, and navigation tools.

Get Started Installation

Key Features

Font Customization

Adjust font size, weight, spacing, and line height to improve readability.

Visual Filters

Apply filters to reduce visual fatigue with brightness, contrast, saturation, and sepia adjustments.

Content Highlighting

Highlight headings and links to improve navigation and content scanning.

Image Controls

Hide images to reduce distractions and focus on text content.

Easy Integration

Simple to implement in any website with minimal configuration required.

Open Source

Free and open source under GPL-3.0 license. Contributions welcome.

Installation

Get started with web-accessibility-plugin in just a few steps via CDN.

1. Include the CDN script

<script src="https://cdn.jsdelivr.net/gh/Cefo14/web-accessibility-plugin@master/release/web-accessibility-plugin.min.cjs.js"></script>

2. Initialize the plugin

<script> window.addEventListener('load', function () { if (typeof window.WebAccessibilityPlugin === 'undefined') return; var container = document.getElementById('root'); var webAccessibilityPlugin = new window.WebAccessibilityPlugin(container); webAccessibilityPlugin.render(); }); </script>
