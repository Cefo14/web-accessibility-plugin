A Simple Accessibility Plugin Alternative

Lightweight, fast and easy to integrate. Enhance web accessibility with customizable text options, visual filters, and navigation tools.

Get Started Installation

Key Features

Font Customization

Adjust font size, weight, spacing, and line height to improve readability for users with visual impairments or reading difficulties.

Visual Filters

Apply filters to reduce visual fatigue and support users with photosensitivity. Includes brightness, contrast, saturation, and sepia adjustments.

Content Highlighting

Highlight headings and links to improve navigation and content scanning for users with cognitive disabilities.

Image Controls

Hide images to reduce distractions, save bandwidth, and help users focus on essential text content.

Easy Integration

Simple CDN-based implementation in any website with minimal configuration. No build tools required.

Open Source

Free and open source under GPL-3.0 license. Community-driven development with contributions welcome.

Installation Guide

Get started with Web Accessibility Plugin in just two simple steps via CDN. No build process or npm packages required.

1. Include the CDN script

<script src="https://cdn.jsdelivr.net/gh/Cefo14/web-accessibility-plugin@master/release/web-accessibility-plugin.min.cjs.js"></script>

2. Initialize the plugin

<script> window.addEventListener('load', function () { if (typeof window.WebAccessibilityPlugin === 'undefined') return; var container = document.getElementById('root'); var webAccessibilityPlugin = new window.WebAccessibilityPlugin(container); webAccessibilityPlugin.render(); }); </script>
